▌ ABOUT THIS PLATFORM

India UTM Command is a research-grade analytics dashboard developed at IIT Bombay for simulating and analyzing Unmanned Traffic Management scenarios across Indian airspace.

Built on real PostgreSQL simulation data, it provides 8 specialized analytical modules covering airspace intelligence, safety conflicts, fleet dynamics, temporal patterns, and ML-based risk prediction.
